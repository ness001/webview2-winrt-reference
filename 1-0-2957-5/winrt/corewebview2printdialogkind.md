# CoreWebView2PrintDialogKind Enum

Specifies the print dialog kind.

| Name |  Value | Description |
|--|--|--|
|`Browser` | 0x0  |  Opens the browser print preview dialog.|
|`System` | 0x1  |  Opens the system print dialog.|
